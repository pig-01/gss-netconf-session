Google Sheets API 設定
說明：

若要從 Google Sheets 載入成員資料及更新資料，請設定以下資訊。

取得 API Key 步驟：

  1. 前往 Google Cloud Console
  2. 建立或選擇專案
  3. 啟用 Google Sheets API
  4. 建立憑證 (API Key)
  5. 將試算表設為「知道連結的使用者」可檢視

設定 Google Apps Script Web App（用於寫入）：

  1. 開啟試算表，點擊「擴充功能」>「Apps Script」
  2. 貼上提供的腳本程式碼（請參考文件）
  3. 部署為「Web 應用程式」
  4. 將 Web 應用程式 URL 貼在下方
